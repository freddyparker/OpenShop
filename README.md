## 📋 Étapes de déploiement

### 1️⃣ Initialiser Git
```bash
cd c:\freddy\projet\curcuma
git init
git add .
git commit -m "Initial commit - Open Shop"
```

### 2️⃣ Créer un repository sur GitHub
- Allez sur [github.com/new](https://github.com/new)
- Créez un nouveau repository `open-shop-curcuma`
- Suivez les instructions pour pousser votre code

### 3️⃣ Déployer sur Vercel
**Option A : Via GitHub (recommandé)**
1. Allez sur [vercel.com](https://vercel.com)
2. Cliquez "New Project"
3. Sélectionnez votre repository GitHub
4. Cliquez "Deploy" ✅

**Option B : Vercel CLI**
```bash
npm install -g vercel
vercel
```

## 🌐 Accès
